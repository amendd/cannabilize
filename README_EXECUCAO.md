# 🚀 EXECUTAR O SITE - MODO FÁCIL

## ⚡ Método Mais Rápido (Windows)

### **Opção 1: Duplo Clique (Recomendado)**

1. Vá até a pasta: `C:\Users\Gabriel\clickcannabis-replica`
2. **Duplo clique** no arquivo: `EXECUTAR_AGORA.bat`
3. Aguarde alguns minutos (primeira vez)
4. O navegador abrirá automaticamente em: **http://localhost:3000**

## 📋 O que o script faz automaticamente:

✅ Verifica se Node.js está instalado  
✅ Instala todas as dependências  
✅ Cria arquivo `.env` automaticamente  
✅ Configura SQLite (não precisa PostgreSQL)  
✅ Gera o banco de dados  
✅ Popula com dados de exemplo  
✅ Inicia o servidor

## ⚠️ Se der erro:

1. **Node.js não encontrado:**
   - Instale Node.js de: https://nodejs.org/

2. **Porta 3000 em uso:**
   - Feche outros programas usando a porta 3000
   - Ou edite o script para usar outra porta

3. **Erro de permissão:**
   - Execute como Administrador
   - Clique direito → "Executar como administrador"
